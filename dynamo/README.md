# Dynamo: Аудит остекления по уровням

## Что делает
Считает по каждому уровню:
- кол-во окон (`OST_Windows`) и их суммарную площадь;
- кол-во **стеклянных** панелей витражей (`OST_CurtainWallPanels`, в имени типа есть `glaz/glass/стек`);
- кол-во и площадь **наружных** стен (Function = Exterior);
- WWR = Σ стекло / Σ наружные стены;
- разбивку по типам окон;
- отклонение Σ стекло от эталонного этажа `L11`.

Это закрывает расхождение в таблице: на `L02–L08` остекление = 39–40 шт / ~387 м², на `L11–L25` = 54 шт / 512.6 м² при одинаковом фасаде. Скрипт покажет, **где именно** различие — в семействах окон, типах витражных панелей или классификации стен.

## Как запустить

1. В Revit открыть модель башни.
2. **Manage → Dynamo → New**.
3. Кинуть на холст узел **Python Script**.
4. Открыть его, вставить содержимое `WindowAuditByLevel.py`.
5. Выход узла → `Data.ExportExcel` (или `Excel.WriteToFile` для старого Dynamo):
   - `filePath`: путь к новому .xlsx
   - `sheetName`: `"Glazing audit"`
   - `startRow / startCol`: 0 / 0
   - `data`: выход Python Script
   - `overWrite`: true
6. Run.

## Что искать в результате

- **Δ Glass vs L11, %** > ±5 % — этаж выпадает из типового.
- Если на L02–L08 столбец **Panels (glazed)** = 0, а на L11–L25 — десятки, значит на нижних этажах фасад промоделирован окнами в обычных стенах, а на верхних — витражом. Это объясняет разницу в Revit-подсчёте «Стекло, шт».
- Если у проблемных этажей в **Window Types** появляются типы, которых нет на L11 — это устаревшие/дублирующие семейства.
- Если **Ext.Walls** на L02–L08 = 45, а на L11–L25 = 43 — на нижних этажах два сегмента фасада не помечены как навесная стена (отсюда «Без остекл., шт» = 45 vs 43).

## Что чинить в модели после аудита

| Симптом                                  | Действие                                      |
|------------------------------------------|-----------------------------------------------|
| Разные семейства окон по уровням         | Type Selector → подменить на единый тип       |
| Стена вместо витража                     | Edit Wall Type → Curtain Wall, скопировать сетку с эталонного этажа |
| Панели не помечены как Glazed            | В типе панели задать материал Glass / переименовать |
| L14: лишние +2 кВт                       | Найти лишний оконный экземпляр на этаже       |

## Зависимости
Только стандартные ноды Dynamo + Revit API. Тестировалось на Revit 2022–2024.
Для Revit < 2021 в `to_m2/to_m` сработает ветка через `DisplayUnitType` — раскомментируйте при необходимости.
